Usage

1. It is necessary you have defined on Matlab the X - data matrix. Size of
matrix (X) must be n-by-p; n=observations, p-independent variables=columns. 

2. For running this file it is necessary to call the rafisher function as 
HZmvntest(X,c,alpha). Please see the help HZmvntest.

3. Once you input the arguments, it will appears your results.
